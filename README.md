# 🕌 رواق المسلم - متجر إلكتروني
## Riwaqi Muslim Store - E-commerce with Django

متجر إلكتروني متكامل لبيع الملابس والكتب الإسلامية مع لوحة تحكم إدارية كاملة.

---

## 🚀 الميزات

### المتجر (Frontend)
- 🏠 صفحة رئيسية مع عروض خاصة ومنتجات جديدة
- 🛍️ عرض جميع المنتجات مع فلترة حسب الفئات
- 🔍 بحث عن المنتجات
- 📄 صفحة تفصيلية لكل منتج مع معرض صور
- 🛒 سلة مشتريات كاملة (إضافة/حذف/تعديل الكمية)
- 💳 صفحة إتمام الطلب مع خيارات الدفع
- 📦 تتبع الطلبات برقم الهاتف
- 📱 تصميم متجاوب لجميع الأجهزة
- 🌐 واجهة عربية كاملة (RTL)

### لوحة التحكم (Admin Panel)
- 📊 إدارة المنتجات (إضافة/تعديل/حذف)
- 📁 إدارة الفئات والفئات الفرعية
- 📋 إدارة الطلبات وتغيير حالاتها
- 🖼️ رفع صور متعددة لكل منتج
- 📈 عرض إحصائيات المبيعات
- 🔍 بحث وفلترة متقدمة

---

## 🛠️ التقنيات

- **Backend:** Django 6.0
- **Database:** SQLite (يمكن التغيير إلى PostgreSQL)
- **Frontend:** HTML5, CSS3, JavaScript
- **Font:** Tajawal (Google Fonts)
- **Icons:** Font Awesome 6.4

---

## 📦 التثبيت والتشغيل

```bash
# 1. الدخول إلى مجلد المشروع
cd riwaqi_store

# 2. تطبيق migrations
python manage.py migrate

# 3. إنشاء مستخدم مدير
python manage.py createsuperuser

# 4. تحميل البيانات التجريبية (اختياري)
python manage.py populate_data

# 5. تشغيل السيرفر
python manage.py runserver
```

## 📁 هيكل المشروع

```
riwaqi_store/
├── riwaqi_store/          # إعدادات المشروع
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
├── core/                  # التطبيق الأساسي
├── products/              # إدارة المنتجات
│   ├── models.py          # Category, Product, ProductImage
│   ├── admin.py           # لوحة تحكم المنتجات
│   ├── views.py           # عرض المنتجات
│   └── management/
│       └── commands/
│           └── populate_data.py  # تحميل البيانات
├── orders/                # إدارة الطلبات
│   ├── models.py          # Order, OrderItem, Cart
│   ├── admin.py           # لوحة تحكم الطلبات
│   ├── views.py           # السلة والطلبات
│   └── context_processors.py
├── templates/             # القوالب
│   ├── base.html          # القالب الأساسي
│   ├── core/home.html     # الصفحة الرئيسية
│   ├── products/          # قوالب المنتجات
│   └── orders/            # قوالب الطلبات
├── static/                # الملفات الثابتة
│   ├── css/style.css      # التنسيقات
│   └── js/main.js         # JavaScript
└── manage.py
```
